# 発電量構成比

【定義】
発電量構成比とは、一定期間に発電された電力量のうち、
各電源（発電に用いられるエネルギー源）が占める割合を示した指標である。
発電量構成比は、電力がどのような電源から供給されているかを表す。

【グラフとどう関係するか】
EneChartでは、年度ごとの発電量構成比を棒グラフとして表示している。
EneChartのグラフは、各電源の発電量の割合を比較することを目的としている。

【読み取れること】
発電量構成比からは、
どの電源が多く利用されているかを読み取ることができる。
また、電源の安定性や環境負荷の傾向を考える手がかりとなる。

【読み取れないこと】
発電量構成比は、
エネルギー自給率や発電コストを直接示すものではない。
また、将来の発電計画や政策の意図を示す指標ではない。

【誤解しやすい点】
発電量構成比が高い電源が、
必ずしも環境に優しいとは限らない。
また、発電量構成比が低い電源でも、
電力の安定供給に重要な役割を担っている場合がある。
