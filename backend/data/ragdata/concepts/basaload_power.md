# ベースロード電源

【定義】
ベースロード電源（電力供給の土台となる電源）とは、
一日を通して安定的に電力を供給する役割を担う電源である。
ベースロード電源は、電力需要が少ない時間帯でも継続して運転される。

【特徴】
ベースロード電源は、発電量の急激な増減が難しい代わりに、
長時間にわたって一定量の電力を供給できる。
このため、電力システム全体の安定性を支える役割を持つ。

【代表的な電源】
日本では、原子力、石炭、大規模な水力発電が、
ベースロード電源として利用されてきた。

【グラフとどう関係するか】
EneChartの発電量構成比では、
ベースロード電源に該当する電源は、
一定の割合を占める傾向がある。
これは、ベースロード電源が電力供給の基盤として機能しているためである。

【誤解しやすい点】
ベースロード電源は、環境負荷が低い電源を指す言葉ではない。
また、ベースロード電源であることが、
将来にわたって同じ役割を担い続けることを意味するわけではない。
